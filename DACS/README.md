# Dự án Full-stack (React + Next.js + MySQL)

Cấu trúc dự án:
- `/frontend`: Ứng dụng React (Vite/TS).
- `/backend`: API server sử dụng Next.js.
- `/database`: Cấu hình và script SQL cho MySQL.

## Cách bắt đầu (Tương lai)
1. Cấu hình cơ sở dữ liệu trong `/database`.
2. Truy cập `/backend` và chạy `npm install`.
3. Truy cập `/frontend` và chạy `npm install`.
